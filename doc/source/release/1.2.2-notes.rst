=========================
smash 1.2.2 Release Notes
=========================

The smash 1.2.2 release fixes a NumPy compatibility issue:

- ``'np.in1d' has been replaced by 'np.isin'``
    `np.in1d <https://numpy.org/doc/2.2/reference/generated/numpy.in1d.html>`__ is deprecated since version 2.0. `np.isin <https://numpy.org/doc/stable/reference/generated/numpy.isin.html>`__ is used instead.
