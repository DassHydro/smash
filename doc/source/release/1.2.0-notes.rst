=========================
smash 1.2.0 Release Notes
=========================

The smash 1.2.0 release adds several functionalities and fixes some compatibility issues, including:

- ``New IO methods for handling parameters with GeoTiff``
    IO methods have been added to read and write gridded hydrological parameters in GeoTiff format.

- ``Compatibility with f90wrap and Python versions``
    Fixed compatibility issues with a recent version of f90wrap, added support for Python 3.13, and dropped support for Python 3.9.
